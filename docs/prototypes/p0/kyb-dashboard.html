KYB 边界

KYB 只做身份与合规验证输入，不替业务方提交事实。

identity gate
可以看

企业注册信息、受益所有人、账户匹配、制裁/黑名单状态和高风险地区。

可以决定

通过、补材料、拒绝、需要人工合规复核。

不能做

不能审批融资、不能改贸易文件、不能查真实名单或保存真实敏感资料。
