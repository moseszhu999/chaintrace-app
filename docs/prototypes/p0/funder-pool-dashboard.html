Pool provider boundary

资金方可以和合约交互，但不能修改贸易事实或质检事实。

contract role
可以做

查看池子、读取候选资产、承诺流动性、查看模拟交易。

不能做

不能改发票、不能改质检结论、不能替出口商签名。
