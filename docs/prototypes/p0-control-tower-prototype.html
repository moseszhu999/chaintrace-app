P0 Control Tower Prototype

A single-page static prototype showing how ChainTrace turns a cross-border trade financing case into a role-based control tower: exporter financing intent, trade materials, Agent gaps, inspection facts, funder review, DeFi/contract state, platform exceptions, and final funding result.

Prototype Boundary

What this prototype intentionally does not do

This HTML is the first visual skeleton for discussion and validation. It avoids implementation details and protects the product narrative.

No old demo fallback
No proof/hash main page

Proof and hash can become later evidence tools, but they are not the P0 product center.

No generic dashboard

The center is the financing control tower and funding result, not a card wall.

No chat-only AI

Agent appears as task, gap, explanation, summary, escalation, and controlled automation.
